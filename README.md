This is Team-1

Open the project folder in cmd. Run the command 'npm install', to download all the node modules. Now, run the command 'npm install -g json-server', to get global json server. 
Open cmd from the DB folder in src folder in the project folder. Now, run the command 'json-server   --watch   --port 4500   db_details.json'.
Open the localhost link that shows up in cmd.
Now open cmd from the project folder and run 'npm start'.
